# FileLock Pro v2.0.0

เว็บแอป static สำหรับเข้ารหัสและถอดรหัสไฟล์ในเบราว์เซอร์ ข้อมูลไม่ถูกอัปโหลดไปยังเซิร์ฟเวอร์

## ใช้งานบน GitHub Pages

1. อัปโหลด `index.html` ไปยัง repository
2. เปิด Settings → Pages
3. เลือก Deploy from a branch และโฟลเดอร์ `/ (root)`

ไฟล์รูปแบบเดิม `FLPR` ใช้ AES-256-GCM, PBKDF2-SHA-256 100,000 รอบ และ chunk 5 MiB ส่วน v2 ยังคงรูปแบบเดิมเพื่อให้ถอดรหัสไฟล์เก่าได้ โดยสร้างไฟล์ใหม่ทีละ chunk 4 MiB

> สำหรับไฟล์ขนาดใหญ่ควรใช้ Chrome หรือ Edge และยอมรับหน้าต่างเลือกตำแหน่งบันทึก เพื่อให้เขียนผลลัพธ์ลงดิสก์โดยตรงโดยไม่เก็บทั้งไฟล์ใน RAM

## โครงสร้างไฟล์

- `index.html` — หน้าเว็บและกลไกเข้ารหัสหลัก
- `styles.css` — รูปแบบเพิ่มเติม
- `security.js` — ล้างรหัสผ่านและข้อมูล preview เมื่อซ่อน/ปิดหน้า
- `manifest.webmanifest` — ข้อมูลติดตั้ง PWA
- `sw.js` — cache เฉพาะไฟล์ตัวแอป ไม่ cache ไฟล์ของผู้ใช้
- `icon.svg` — ไอคอนแอป

นโยบาย CSP ของหน้าเว็บปิดการเชื่อมต่อออกไปภายนอก (`connect-src 'none'`) และปิดกล้อง ไมโครโฟน ตำแหน่ง USB และ payment API ข้อมูลไฟล์และรหัสผ่านไม่ถูกบันทึกลง localStorage, IndexedDB หรือ service-worker cache
